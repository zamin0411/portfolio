# Hồ Gia Mẫn — Portfolio

A modern, cute pastel portfolio website built with React, Vite, TailwindCSS, and Framer Motion.

## Tech Stack

- **React 18** + **Vite**
- **TailwindCSS** — utility-first styling with custom pastel theme
- **Framer Motion** — smooth animations and transitions

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

- Pastel gradient background (pink → lavender → blue)
- Rounded 2xl cards with soft shadows

- Smooth scroll navigation
- Fully responsive
